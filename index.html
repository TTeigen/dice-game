The goal of this game is to get 2000 or more points by "saving" certain combinations of dice and banking their corresponding point value.

Your turn is bust(over and you gain no points) if none of the following situations apply:

  • One of the dice is a "1"
  • One of the dice is a "5"
  • At least 3 dice have the same value

Scoring:

  • Singles a "1" is worth 100 points, a "5" is worth 50
  • 3 of a Kind 3 of any # except "1" is worth 100 x the face value (3 = 300), 1 = 1000
  • 3 of a Kind 3 of any # except "1" is worth 200 x the face value (3 = 600), 1 = 2000
  • 3 of a Kind 3 of any # except "1" is worth 400 x the face value (3 = 1200), 1 = 3000
